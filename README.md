# Projeto Previsão do Tempo dos Últimos Anos no Rio de Janeiro

### Resumo do projeto
Nesse projeto pegamos uma base de dados com os registros da previsão do tempo dos últimos anos no estado do Rio de Janeiro. O objetivo foi analisar o Dataset e com ele gerar um modelo de previsão utilizando **Gradient Boosting**. Além da análise dos dados, foi feito a converssão para uma série temporal, gráficos, separação dos dados em treino e teste, avaliação dos dados com as métricas de **R2** e **Mape** presentes no pacote do **Scikit_Learn**.
